An XSS injection (or Cross-site Scripting) is an attack in which a hacker inputs a piece of code that the webapp or interface deems legitimate, but will actually cause changes in the website's source code.

There are 3 kinds of XSS. Reflected XSS which works on the user in one go, Persistent XSS which causes permanent or lasting changes, and DOM XSS, which causes physical defects on the interface.
